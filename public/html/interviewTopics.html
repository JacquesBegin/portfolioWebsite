Interview Topics

JavaScript values that have a "falsey" value. That is, these values have an inherit false value when tested in a conditional statement.
  • false - Boolean
  • 0 - Number
  • "" - Empty String
  • undefined - A variable that is declared but not defined with a value.
  • NULL - An empty value that can be assigned to a variable.
  • NAN - A value that is not a number. The value of NAN is never consistent, therefore equality cannot be checked with NAN === NAN.
