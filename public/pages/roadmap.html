Web Developer Roadmap 2019

As I continue growing my Web Developer career, I find there are an ever increasing number of technologies and skills required to be proficient in the field. It can be difficult, if not overwhelming, to understand what is necessary for becoming a capable Web Developer. Kamran Ahmed has developed a detailed roadmap for navigating the Web Developer path.

I am using this road map as my guide while making adjustments for personal interests and ultimate goals.

The following sections will layout my roadmap and assign skill levels for each step in the path. As I gain more knowledge and proficiency in a given subject, skill levels will be increased.

  • Level 0 - Unfamiliar/No Previous Knowledge
  • Level 1 - Understands Purpose/Basic Use Cases
  • Level 2 - Can Use Technology/Conpects At A Basic Level
  • Level 3 - Comfortable With Technology While Using References
  • Level 4 - Strong Use of Technology/Concepts With Limited Reference Use
  • Level 5 - Capable of Using/Explaining Without Using References
